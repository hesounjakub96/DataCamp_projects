# Predicting Movie Rental Durations

## Objective
Build a regression model to predict how long customers rent DVDs based on transaction and metadata features.

## Data Preparation
- Created target variable (rental duration in days)
- Engineered binary features from special features column

## Analysis
- Calculated rental duration from date differences
- Split dataset into training and testing sets
- Trained decision tree regressor model to predict rental duration
- Evaluated models using Mean Squared Error (MSE)
