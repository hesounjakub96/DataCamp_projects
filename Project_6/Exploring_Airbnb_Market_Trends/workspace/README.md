# Exploring Airbnb Market Trends

## Objective
Analyze Airbnb listing data in New York to understand pricing, room types, and review activity in the short-term rental market.

## Data 
- Imported data from multiple file formats (CSV, Excel, TSV)
- Standardize categorical data to consistent labels

## Analysis
- Identified earliest and most recent review dates
- Counted the number of private room listings
- Calculated average listing price
