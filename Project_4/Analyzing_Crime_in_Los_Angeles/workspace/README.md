#  Analyzing Crime in Los Angeles

## Data
The original data set is over 25MB. Uploaded version is only sample with size 50000.

##  Objective
Analyze crime data in Los Angeles to identify temporal patterns, high-risk locations, and demographic trends among victims.

##  Dataset
Crime dataset containing information about incident time, location, and victim demographics.

##  Analysis
- Identified the hour with the highest frequency of crimes
- Determined the location with the highest number of night crimes
- Grouped victims into age categories and calculated crime distribution
