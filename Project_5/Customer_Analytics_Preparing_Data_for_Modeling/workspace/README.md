# Customer Analytics: Preparing Data for Modeling

## Objective
Prepare customer data by cleaning, transforming, and optimizing the dataset for efficient storage.

## Data Preparation
- Converted binary categorical columns to boolean data types
- Optimized numerical columns (int32, float16) to reduce memory usage
- Transformed nominal categorical variables to category dtype
- Converted ordinal variables into ordered categories
- Filtered dataset based on experience and company size criteria
