#  Exploring NYC Public School Test Results Scores

##  Objective
Analyze NYC public school SAT performance data to identify top-performing schools and explore differences across boroughs using math, reading, and writing scores.

##  Dataset
NYC public school test results dataset containing SAT scores (math, reading, writing) for schools across different boroughs.

##  Analysis
- Identified schools with best math performance (≥ 80% of max score)
- Calculated total SAT score as sum of all sections
- Ranked top 10 schools based on combined SAT score
- Analyzed scores of grouped data of NYC boroughs
