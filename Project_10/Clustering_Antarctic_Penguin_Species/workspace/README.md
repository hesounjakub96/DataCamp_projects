# Clustering Antarctic Penguin Species

## Objective
Perform unsupervised learning to identify natural groupings of penguins based on physical measurements and characteristics using clustering techniques.

## Data Preparation
- Created dummy variable for sex
- Standardized features using StandardScaler

## Analysis
- Applied K-Means clustering to identify 3 groups
- Assigned cluster labels to each observation
- Visualized clusters using key physical measurements
- Computed cluster centroids and transformed them back to original scale
