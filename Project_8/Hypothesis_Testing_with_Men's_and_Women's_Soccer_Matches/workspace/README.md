# Hypothesis Testing with Men's and Women's Soccer Matches

## Objective
Determine whether there is a statistically significant difference in the mean number of goals scored in FIFA World Cup matches between men's and women's teams since 2002.

## Data Preparation
- Loaded men's and women's match datasets
- Filtered data to include IFA World Cup matches from 2002 onwards only

## Analysis
- Computed total goals per match for both datasets
- Performed two-sample t-test (Welch’s t-test)
- Compared mean goals between men's and women's matches
- Evaluated statistical significance at 10% significance level
